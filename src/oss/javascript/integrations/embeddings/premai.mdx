---

The `PremEmbeddings` class uses the Prem AI API to generate embeddings for a given text.

## Setup

In order to use the Prem API you'll need to [sign up for a Prem account and create an API key](https://studio.premai.io/sign-up).

Then install the [`@langchain/community`](https://www.npmjs.com/package/@langchain/community) package:

<Tip>
See [this section for general instructions on installing LangChain packages](/oss/langchain/install).
</Tip>

```bash npm
npm install @langchain/community @langchain/core
```

## Usage

```typescript
import { PremEmbeddings } from "@langchain/community/embeddings/premai";

const embeddings = new PremEmbeddings({
  // In Node.js defaults to process.env.PREM_API_KEY
  apiKey: "YOUR-API-KEY",
  // In Node.js defaults to process.env.PREM_PROJECT_ID
  project_id: "YOUR-PROJECT_ID",
  model: "@cf/baai/bge-small-en-v1.5", // The model to generate the embeddings
});

const res = await embeddings.embedQuery(
  "What would be a good company name a company that makes colorful socks?"
);
console.log({ res });
```
